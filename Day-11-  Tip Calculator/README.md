# 💸 Tip Calculator 🧾

A simple JavaScript project that helps users quickly calculate tips and split bills.  
The app displays the **tip amount per person and total per person** while providing an easy-to-use interface.

---

## 🚀 Features

- Calculate tip based on:
  - **Bill amount**
  - **Tip percentage** (select or custom)
  - **Number of people sharing the bill**
- Display:
  - **Total bill**
  - **Total per person**
- Reset functionality to start over
- Minimal, mobile-friendly design
